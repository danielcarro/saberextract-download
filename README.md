# SaberExtract

Aplicativo desktop para Windows que extrai contatos de grupos do WhatsApp Web, organiza os contatos em listas e permite exportar para CSV, XLS e PDF.

## Download

Baixe o instalador mais recente:

[Baixar SaberExtract_Setup_v1.0.0.exe](https://raw.githubusercontent.com/danielcarro/saberextract-download/main/SaberExtract_Setup_v1.0.0.exe)

## Requisitos

- Windows 10 ou 11
- Google Chrome instalado
- Conexao com a internet

## Instalacao

1. Baixe o arquivo `SaberExtract_Setup_v1.0.0.exe`.
2. Execute o instalador.
3. Abra o SaberExtract.
4. Escaneie o QR Code do WhatsApp Web quando solicitado.
